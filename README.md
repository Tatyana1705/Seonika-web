# Проект сайта "SEO-web"

## Установка и запуск

- Устанавливаем npm пакеты:

```npm ci```

- Запуск проекта для разработки

```npm run start```

- Сборка проекта, проект соберентся в паку build

```npm run build```

- Запуск stylelint

```npm run test```
